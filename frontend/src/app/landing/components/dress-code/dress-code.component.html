Dress Code Guidelines

Изберете елегантно облекло, което съответства на официалния характер на събитието. За дамите препоръчваме коктейлни рокли, а за господата костюми в светли тонове.

Do's & Don'ts:

  • ✓ Носете светли и елегантни тонове
  • ✓ Изберете удобни обувки за танци
  • ✗ Избягвайте прекалено спортно облекло
